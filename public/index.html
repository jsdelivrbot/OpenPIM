OpenPIM

  • Public key (PEM)

    openssl rsa -in nsantos.pem -pubout -out nsantos.pub

  • Encrypted Private key (PEM)

    openssl genpkey -algorithm RSA -out nsantos.pem -aes-256-cbc -pkeyopt rsa_keygen_bits:4096

  • Private key (PEM)

    openssl rsa -in nsantos.pem -out nsantos.priv

  • Notes

    echo -n "secret" | openssl pkeyutl -encrypt -pubin -inkey nsantos.pub | openssl base64 -A
